## 📥 WHAT YOU NEED TO DOWNLOAD & CONFIGURE

### 1. **Google Services JSON** (Android)
```
📥 Download from: Firebase Console → Project Settings → Download google-services.json
📂 Place at: android/app/google-services.json
🔧 Required for: Android authentication & Firebase services
⏱️ Time: 2 minutes
```

**Steps:**
1. Go to https://console.firebase.google.com/
2. Select project: "questkids-mobile"
3. Click ⚙️ (Settings) → Project Settings
4. Scroll down → "Your apps" section
5. Find Android app "com.questkids.questkids"
6. Click "Download google-services.json"
7. Save to `android/app/google-services.json`

### 2. **iOS Service File** (Already Present)
```
✅ Already at: ios/Runner/GoogleService-Info.plist
🔧 No action needed (was auto-generated by FlutterFire)
```

### 3. **Gmail App Password** (For SMTP)
```
🔐 Create at: https://myaccount.google.com/apppasswords
📧 Account: questkids.dev@gmail.com
⏱️ Time: 5 minutes
```

**Steps for Gmail App Password:**
1. Enable 2-Factor Authentication first:
   - Go to https://myaccount.google.com/security
   - Find "2-Step Verification"
   - Enable it

2. Create App Password:
   - Go to https://myaccount.google.com/apppasswords
   - Select: "Mail" + "Windows Computer" (or your device)
   - Google generates 16-character password
   - **Copy this password** (you'll use it in step 4 below)

### 4. **Deploy Cloud Functions**
```
📂 Location: functions/src/index.ts ✅ Created
🎯 Purpose: Send SMTP emails automatically
⏱️ Time: 10 minutes
```

**Steps:**
1. Open Terminal in VS Code
2. Install Firebase CLI (if not done):
   ```bash
   npm install -g firebase-tools
   firebase login
   ```

3. Navigate to functions directory:
   ```bash
   cd functions
   npm install
   ```

4. Set SMTP credentials:
   ```bash
   firebase functions:config:set \
     mail.sender="questkids.dev@gmail.com" \
     mail.password="[your-16-char-app-password-here]"
   ```

5. Deploy:
   ```bash
   cd ..
   firebase deploy --only functions
   ```

6. Verify deployment:
   ```bash
   firebase functions:log
   ```

### Firestore Security Rules
```
📁 Location: Firestore Database → Rules
🔧 Apply these rules to protect user data:
```

```typescript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Users can only read/write their own data
    match /users/{userId} {
      allow read, write: if request.auth.uid == userId;
    }
    
    // Activities readable by authenticated users
    match /activities/{document=**} {
      allow read: if request.auth != null;
      allow write: if false;
    }
    
    // Progress entries authenticated users
    match /progress/{document=**} {
      allow read, write: if request.auth != null;
    }
    
    // Rewards user-specific
    match /rewards/{userId} {
      allow read, write: if request.auth.uid == userId;
    }
    
    // Notifications user-specific
    match /notifications/{document=**} {
      allow read, write: if request.auth != null;
    }
    
    // Emails collection (written by app, read by functions)
    match /emails/{document=**} {
      allow write: if request.auth != null;
      allow read: if false; // Functions access via admin SDK
    }
  }
}
```

## 🔒 SECURITY REMINDERS

1. **Never commit secrets to Git**
   - Gmail App Password is stored in Firebase Functions only
   - Use `.gitignore` for `.env` files

2. **Firestore Rules are Critical**
   - Users can only read/write their own data
   - Activities are read-only by students
   - Admin operations use Cloud Functions

3. **Email Verification**
   - Require email verification for sensitive operations
   - Implement parent consent for grade 4 learners

4. **Password Security**
   - Minimum 8 characters required
   - Firebase handles password hashing

5. **Rate Limiting**
   - Implement rate limiting for login attempts
   - Prevent brute force attacks

## 🐛 TROUBLESHOOTING

### Android google-services.json Issues
```
Error: "google-services.json not found"
Fix: Ensure file is at android/app/google-services.json (not in a subfolder)
```

### Cloud Functions Not Sending Emails
```
Check logs: firebase functions:log
Common issues:
- Gmail App Password is wrong
- 2FA not enabled on questkids.dev@gmail.com
- SMTP credentials not set correctly
```

### Google Sign-In Not Working
```
Android: Register SHA-1 fingerprint in Firebase
  Run: ./gradlew signingReport (in android/)
  Add to Firebase Console: Project Settings → Your apps

iOS: Verify CFBundleURLSchemes in Info.plist
```

### Phone OTP Not Sending
```
1. Enable RECAPTCHA in Firebase Console
2. Add test phone numbers if in testing mode
3. Check Firebase Phone Auth quotas
```
